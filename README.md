## 📋 Sobre o Projeto

O **Travelgram** é uma página de perfil inspirada em redes sociais de viagem, onde são exibidas informações do usuário, estatísticas e uma galeria de fotos. O projeto foi desenvolvido com foco em **CSS Flexbox**, explorando alinhamentos, distribuição de espaço e organização de componentes de forma fluida e responsiva.

---

## ✨ Funcionalidades

- ✅ Página de perfil com foto, bio e localização do usuário
- ✅ Estatísticas de viagens (países visitados, fotos, seguidores)
- ✅ Galeria de fotos organizada com espaçamento uniforme
- ✅ Navegação com ícones e links de ação
- ✅ Tipografia hierárquica e legível

## 🧠 Foco Técnico — CSS Flexbox

Este projeto foi usado como laboratório para o domínio do Flexbox. Alguns conceitos aplicados:

- `display: flex` com `align-items` e `justify-content` para alinhamentos precisos
- `flex-direction` para alternar entre layouts em linha e coluna
- `gap` para espaçamento uniforme entre os elementos da galeria
- `flex-shrink` para evitar distorção de imagens em telas menores
- Composição de componentes independentes com Flexbox aninhado
